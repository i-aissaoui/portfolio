# 🕋 Hadj Management System

> **A centralized logistics and medical scheduling platform for pilgrim groups.**

## 🎯 Project Overview
Built to handle the complex logistics of the Hadj pilgrimage, this platform manages transportation, hotel assignments, and medical appointment tracking for large groups of pilgrims.

## 🛠️ Tech Stack
• **Frontend:** React.
• **Backend:** FastAPI, PostgreSQL.
• **Features:** Dynamic scheduling, automated reporting, and medical record management.
