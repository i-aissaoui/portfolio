# 📅 Intelligent University Scheduling System

> **Solving complex Timetabling Problems using Swarm Intelligence and Genetic Algorithms.**

## 🎯 Project Overview
Automating university schedules is a "Hard Constraint" problem (NP-Complete). This project uses hybrid optimization algorithms to schedule classes, teachers, and rooms without conflicts.

## 🧠 Optimization Algorithms
• **Genetic Algorithm (GA):** Evolves schedules through crossover and mutation.
• **Particle Swarm Optimization (PSO):** Models schedule search as particles moving through a constraint space.
• **Ant Colony Optimization (ACO):** Finds optimal paths for resource allocation.
• **Simulated Annealing:** Escapes local minima by probabilistic "cooling."

## 🛠️ Tech Stack
• **Engine:** Python, NumPy, Pandas.
• **Frontend:** Next.js 14, TypeScript.
• **Backend:** FastAPI.
