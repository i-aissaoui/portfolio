# PhD Open Position: CESI LINEACT (Strasbourg) - eTruckCharge Project

## Project Title
Federated Graph Neural Networks for Intrusion Detection in electric vehicle networks.

## Supervisors
- **Dr. Amine BRAHMIA** (abrahmia@cesi.fr)
- **Dr. Mourad ZGHAL** (mzghal@cesi.fr)
- **Dr. Zakariya GHALMANE** (zghalmane@cesi.fr)

## Key Technical Focus
1. **Federated Learning**: Distributed intrusion detection for EV fleets.
2. **Graph Neural Networks (GNN)**: Modeling vehicle-to-infrastructure communication as graphs.
3. **Complex Network Theory**: Using centrality measures to identify critical nodes in the network.
4. **Cybersecurity**: Attack patterns like spoofing, DoS, and injection in EV systems.

## Strategy
This position is a **perfect match** because of your **Career Connect** project (GNN expertise) and your **Sonatrach** project (Complex CPS architectures). Emphasize that you have already built GNN models for relational data and that you understand the "structural dependencies" requested in the offer.
