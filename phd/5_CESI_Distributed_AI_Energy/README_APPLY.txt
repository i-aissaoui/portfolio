# PhD Application: CESI LINEACT (Strasbourg) - FAIR'nRG Project

## Project Title
Intelligence artificielle distribuée pour la performance énergétique collaborative des prosommateurs : application à l'optimisation énergétique des systèmes PV+stockage dans le Rhin Supérieur.

## Supervisors
- **Dr. Youssef JOUANE** (yjouane@cesi.fr)
- **Dr. Amine BRAHMIA** (abrahmia@cesi.fr)
- **Dr. Jean-François DOLLINGER** (jfdollinger@cesi.fr)

## Key Technical Focus
1. **Federated Learning (Apprentissage Fédéré)**: Heterogeneous data clustering, model specialization.
2. **Dynamic Optimization**: Heuristics/Metaheuristics (PSO, Genetic Algorithms) for storage life and self-consumption.
3. **Incremental Learning**: Continuous model refinement without retraining.
4. **Energy Systems**: PV, Batteries, IoT sensors.

## Strategy
Highlight the **Sonatrach Digital Twin** as a "Cyber-Physical System" (CPS) and emphasize the **Metaheuristics** project for the second axis of the thesis. The work on **Mamba-SSM** and **xLSTM** demonstrates the "distributed/constrained device" performance requested.
