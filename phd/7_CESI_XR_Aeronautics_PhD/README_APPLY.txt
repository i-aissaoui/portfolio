# PhD Open Position: CESI LINEACT (Pau) - Campus Aero Adour (C2A)

## Project Title
Semantic Data Compression for Extended Reality in Aeronautic Industry.

## Supervisors
- **Dr. Samir OUCHANI** (souchani@cesi.fr)
- **Dr. Hugues Marie KAMDJOU** (hmkamdjou@cesi.fr)

## Key Technical Focus
1. **Extended Reality (XR)**: 3D scene streaming, HMD (Head-Mounted Displays) constraints.
2. **Semantic Data Compression**: Encoding meaningful structures (objects, parts) rather than raw pixels.
3. **Edge-Cloud Orchestration**: Offloading semantic inference to edge servers to reduce motion-to-photon latency.
4. **Digital Twins**: Interaction between real systems and their virtual counterparts in aeronautics.
